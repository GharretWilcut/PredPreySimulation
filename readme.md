
# Predator-Prey Simulation

A dynamic simulation modeling the interactions between predator and prey populations.

## Features

- Real-time population dynamics
- Configurable simulation parameters
- Visual representation of predator/prey interactions

## Getting Started

### Prerequisites
need to install 
```bash
sudo apt install libsfml-dev
```

### Installation

```bash
git clone https://github.com/gharret/PredPreySimulation.git
cd PredPreySimulation
```

### Usage

```bash
make 
./predprey
```

## How It Works

The simulation models population changes based on:
- initial population
- repopulation rate
- predator hunger replenishment

## Configuration

Edit the parameters in `config.py` to adjust:
- Initial population counts
- Growth rates
- world size
